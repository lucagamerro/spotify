# UI/UX of Spotify
This is a UI/UX base of the famous Spotify app, made in Vuejs.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
